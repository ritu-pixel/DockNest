# DockNest

DockNest is a collection of containerized applications and tools designed to streamline deployment, orchestration, and management of various services using Docker, Kubernetes, and cloud platforms.

## 📌 Projects Overview

### 1️⃣ **Containerized SQL**
   - A fully containerized SQL database setup using Docker for easy deployment and management.

### 2️⃣ **Docker Bake**
   - Uses Docker BuildKit to speed up and optimize the Docker image-building process.

### 3️⃣ **Docker Bridge Networking**
   - Demonstrates container networking using Docker Bridge to allow inter-container communication.

### 4️⃣ **Docker Swarm - Microservices**
   - Implements a microservices architecture using Docker Swarm for scalable and resilient service orchestration.

### 5️⃣ **Docker Volume Persistence**
   - Shows how to use Docker volumes for persistent storage across container restarts.

### 6️⃣ **Dockerized Streamlit Development Environment**
   - A pre-configured Streamlit development setup inside a Docker container for hassle-free app deployment.

### 7️⃣ **Evidently AI**
   - Integrates Evidently AI for real-time model monitoring and data drift detection in machine learning applications.

### 8️⃣ **MiniKubes and Kubectl**
   - A guide to setting up and managing Kubernetes clusters using Minikube and Kubectl.

### 9️⃣ **Minikube Microservices**
   - Runs microservices on Minikube, demonstrating Kubernetes-based service management.

### 🔟 **Python Logging in Docker**
   - Configures structured logging for Python applications running inside Docker containers.

### 1️⃣1️⃣ **Streamlit on EC2**
   - Deploys a Streamlit app on AWS EC2 using Docker for cloud-hosted analytics and visualization.

### 1️⃣2️⃣ **Streamlit Hello World**
   - A basic "Hello World" Streamlit app demonstrating Dockerized deployment.

### 1️⃣3️⃣ **SQL-PostgreSQL**
   - Implements a PostgreSQL database inside a Docker container with pre-configured settings.

### 1️⃣4️⃣ **Streamlit App**
   - A generic containerized Streamlit application for interactive data visualization and ML model serving.

### 1️⃣5️⃣ **Titanic Prediction Model**
   - A machine learning model wrapped in a Docker container for predicting Titanic survivors.

## 🚀 Installation & Usage

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/ritu-pixel/DockNest.git
   cd DockNest
   ```

2. **Run a Project:**
   ```bash
   cd <project-folder>
   docker-compose up -d
   ```

3. **Stop a Running Project:**
   ```bash
   docker-compose down
   ```
